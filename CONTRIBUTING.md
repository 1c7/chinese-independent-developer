# 如何写好一句话产品介绍

提交项目时，介绍语是别人对你产品的第一印象。

## 口诀

> **用户是谁 + 解决什么问题 + 凭什么相信你**

"凭什么相信你"可以是具体数字、替代谁、独特做法，三选一即可。

## 套模板

```
[目标用户]用的[产品类型]，[核心价值]，[一个可信细节]
```

不需要每个都有，能答出其中两项就够了。

## 真实案例对比

以下好例子均来自本列表：

| ❌ 只说品类（差） | ✅ 说清价值（好） |
|------|------|
| `AI 生图工具` | `AI 语音克隆平台，3 秒克隆任何声音，支持 10 种语言，提供 50+ 预设声音库` |
| `AI 视频生成工具` | `基于 Seedance 2.0，原生多镜头叙事、音视频同步和 2K 影院级画质输出` |
| `书签工具` | `替代 Chrome 新标签页的书签管理器，支持文件夹卡片视图、拖拽排序、多选批量操作` |
| `数据工具` | `现代化 AI 原生数据工作台，支持 MySQL/Postgres/SQLite，Dbeaver / Navicat 替换` |

## 自测

把介绍语给一个不了解你产品的朋友看，问：「这是给谁用的？解决什么问题？」
如果他答不上来，就重写。
