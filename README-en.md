## Chinese Independent Developers' Projects List  
Collect basic information about all Chinese indie devs' ("Independent Developers'") projects.

<br/>

**1. Why does this list exist?**     
As a developer, I am curious about projects that other developers are working on in their spare time.

Every developer is welcome to add their own projects onto this list.
Please make a pull request or a GitHub issue to add your projects.

The project must be a web site or an app (e.g., it cannot be a developer tool or forum web site).
If you are unsure whether your project meets this list's criteria, please contact me.

<br/>
